# Instructions

Check if a given string is a palindrome. Return `true` if the string reads the same backward as forward, and `false` otherwise.

---

<Row>
  <Col>
    <p>The example solution provided demonstrates a function called <code>isPalindrome</code> that checks if a given string is a palindrome. The function converts the string to lowercase. It then compares the cleaned string to its reverse to determine if it reads the same backward as forward.</p>
  </Col>
  <Col>

    ```javascript {{ title: "Check Palindrome" }}
      const isPalindrome = (str) => {
        const cleanedStr = str.toLowerCase();
        const reversedStr = cleanedStr.split('').reverse().join('');
        return cleanedStr === reversedStr;
      }
    ```
  </Col>
